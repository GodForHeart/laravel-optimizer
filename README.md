# [LaravelOptimizer](https://www.juli-jianzhan.com)

📦 一个 PHP 日志服务 SDK。

## 环境需求

- PHP >= 7.2

## 安装

```bash
composer require godforheart/laravel-optimizer
```

## 生成 config 配置文件及迁移文件（数据库日志类型使用）
```bash
php artisan optimizer:publish
```
